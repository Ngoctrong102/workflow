# Infrastructure Directory

This directory contains infrastructure as code (IaC) configurations, deployment scripts, and infrastructure-related resources.

## Directory Structure

- `docker/` - Docker-specific configurations and scripts
- `kubernetes/` - Kubernetes manifests and configurations (if applicable)
- `scripts/` - Infrastructure automation scripts
- `monitoring/` - Monitoring configurations (Prometheus, Grafana, etc.)
- `logging/` - Logging configurations (ELK, Loki, etc.)

## Usage

This directory is managed by the DevOps Expert. See `.cursor/rules/devops-expert.md` for guidelines.
